How to create own theme?

Create 2 new files - NameOfYourTheme_Dark.json and NameOfYourTheme_Light.json. Then, copy and paste in files this -

{
    "WindowText" : "",
    "Button" : "",
    "Light" : "",
    "Midlight" : "",
    "Dark" : "",
    "Mid" : "",
    "Text" : "",
    "BrightText" : "",
    "ButtonText" : "",
    "Base" : "",
    "Window" : "",
    "Shadow" : "",
    "Highlight" : "",
    "Highlighted Text" : "",
    "Link" : "",
    "LinkVisited" : "",
    "AlternateBase" : "",
    "NoRole" : "",
    "ToolTipBase" : "",
    "ToolTipText" : ""
}

Then fill it with RGB colors. Place this files in "Themes" folder, and "ThemesReader.py" file add theme themself! But, how to publish your theme?

How to publish your theme?

DM me on Discord (@timpo14) with message like 'Hi, can i publish own theme on LOVE Constructor? My nickname - (your niickname)', or similar. Then, when you get message like 'Okay, i give you acess' visit GitHub respisitory, and commit your theme in branch fanmade-themes. Then create pull request, and we will accept it if theme does not cause a gag reflex) Before accept, we will add link on your account in your JSONs.
